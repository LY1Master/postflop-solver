# 各(牌面,桶,位置,SPR)所需的修正因子

## 修正公式结构

```
修正后 DLEv = DLEv × α(牌面,桶,位置) × (1 + k(牌面,桶,位置) × (t − 1))

其中 t = 2×SPR/(SPR+5)
      SPR=5 时 t=1.0
      SPR=20 时 t=1.6
```

**说明：**
- **α（基础修正系数）**：SPR=5 时的修正因子，从数据直接得出
- **k（SPR敏感度）**：修正因子随 SPR 变化的程度。k≈0 表示不依赖 SPR（多数情况如此）；k>0 表示深筹码需更大修正；k<0 表示深筹码需更小修正
- 当 k 很小时（|k|<0.15），可忽略 SPR 调整，直接用 α

## 结论

### 1. SPR 影响远小于预期

超过 **80%** 的 (牌面,桶,位置) 组合的 k 值在 [-0.15, +0.15] 范围内，即 SPR 从 5 到 20 的变化对修正因子的影响不超过 10%。**可以忽略 SPR 调整**，仅使用固定修正因子 α。

### 2. 真正需要 SPR 关注的少数情况

仅以下 3 个组合的 k 值显著偏离 0：
- **三条面 IP 桶1**: k=+1.23（深筹码时超强成牌隐含赔率暴增）
- **纯彩虹 IP 桶1**: k=+0.40
- **两同花A/B IP 桶1**: k=+0.31~+0.33

这些共同特征是：**IP 位置的超强成牌在深筹码下隐含赔率暴增**，这是符合扑克理论的。

### 3. 建议的 v4 修正结构

```
修正后 DLEv = DLEv × α(牌面, 桶, 位置)

（大多数情况不需要 SPR 调整）
```

α 值直接使用上表"α(基础)"列。

对于那 3 个特殊情况，额外加 SPR 项：
```
桶1 IP 在三条面/纯彩虹/两同花面:  修正后 DLEv = DLEv × α × (1 + 0.6 × k × (t - 1))
                                其中 t = 2×SPR/(SPR+5)
```
