这份多牌面泛化验证报告非常关键，它暴露了线性外推（Linear Extrapolate）在处理极端 SPR 时的致命缺陷。

### 核心痛点诊断

你指出的原因非常精准：在筹码极深的场景下（$SPR=20$），原本为中等筹码（$SPR \approx 4 \sim 6$）设计的系数 $k$，在乘法公式 $1.0 + k \times SPR$ 中引发了**误差的级数级放大**。
此外，从分桶明细数据来看（例如 `Mean DL` 与 `Mean DL修正` 完全一样），你目前代码在执行时，由于条件分支判定（比如硬编码判定了具体手牌 `AsKc`、特定花色或位置符号不匹配），导致**大部分边缘成牌和空气牌并没有成功勾中修正逻辑，而勾中修正逻辑的部分则被高估或低估了数倍**。

### 工业级修正公式升级（SPR 非线性平滑方案）

为了让 8 种牌面的修正模型既能在 $SPR=2$（短筹码/3Bet/4Bet圈）生效，又能在 $SPR=20$（100BB单次加注圈）保持稳定，我们必须将原先的线性乘数改为**具有饱和上限的非线性函数**，或直接引入**反比对准机制**。

推荐使用**双曲正切函数（Tanh）衰减**或**倒数饱和函数（Reciprocal Saturation）**。为了保证工程落地简单且完全不失真，这里采用最稳健的 **“临界 SPR 对准与边际递减模型”**。

新公式的底层逻辑是：

1. 设定基准 $SPR_{base} = 5$（这是之前的系数敏感区）。
2. 当 $SPR > 5$ 时，其影响不再线性膨胀，而是随深度增加而“边际递减”。
3. 动态 SPR 调节因子公式统一调整为：

$$\text{SPR\_Factor}(SPR) = \frac{2 \times SPR}{SPR + 5}$$


* 当 $SPR = 5$ 时，$\text{SPR\_Factor} = 1.0$（完美继承原先的合理修正区间）。
* 当 $SPR = 20$ 时，$\text{SPR\_Factor} = \frac{40}{25} = 1.6$（自动收敛，杜绝了原先膨胀到 4~7 倍的惨剧）。

## 🛠️ 重新设计：8 种牌面通用乘法修正代码（GTO 生产环境级）

这里不再使用脆弱的手牌字符串判定（避免因大小写、花色符号不一致导致不触发），而是完全基于你数据中提供的**位置、大桶分类、Draw类型**这三个标准标签进行纯矩阵乘法修正：

```python
import math

def get_spr_factor(spr):
    """
    非线性SPR平滑因子：确保深筹码场景下修正值不爆炸
    SPR=5  -> Factor=1.0
    SPR=20 -> Factor=1.6
    """
    return (2.0 * spr) / (spr + 5.0)

# ==================== 1. 三条面 (Trips) ====================
def correct_board_1_trips(dlev, position, bucket, draw, spr):
    f_spr = get_spr_factor(spr)
    
    # 数据表明：原始 DLEv 在弱牌/空气上严重高估 (DL 14~21 -> Full 0.6~4.1)
    if bucket in ["桶6:弱听牌", "桶7:纯空气"]:
        # 弱牌的实现率随 SPR 变深而剧烈恶化（反向隐含赔率）
        return dlev * max(0.1, 1.0 - 0.45 * f_spr)
    
    # 中等成牌（口袋对等）在三条面上极易被两极化范围收割
    if bucket == "桶3:中等成牌":
        return dlev * max(0.5, 1.0 - 0.12 * f_spr)
        
    return dlev

# ==================== 2. 对子彩虹面 (Paired Rainbow) ====================
def correct_board_2_paired_rainbow(dlev, position, bucket, draw, spr):
    f_spr = get_spr_factor(spr)
    
    # OOP 中等成牌（如中对）在干燥面面临长期多街压榨，略微缩水
    if position == "OOP" and bucket == "桶3:中等成牌":
        return dlev * max(0.7, 1.0 - 0.08 * f_spr)
        
    # IP 纯空气由于多街下注垄断优势，在深筹码下变现更优（原先k=0.3太激进，改为k=0.15）
    if position == "IP" and bucket == "桶7:纯空气":
        return dlev * (1.0 + 0.15 * f_spr)
        
    return dlev

# ==================== 3. 对子两同花面 (Paired Two-Tone) ====================
def correct_board_3_paired_twotone(dlev, position, bucket, draw, spr):
    f_spr = get_spr_factor(spr)
    
    # 相比彩虹面，湿润对子面的成牌防守压力更大
    if position == "OOP" and bucket == "桶3:中等成牌":
        return dlev * max(0.65, 1.0 - 0.10 * f_spr)
        
    # 弱牌与空气，如果没有听牌，稍微向下微调
    if bucket in ["桶6:弱听牌", "桶7:纯空气"] and draw == "None":
        return dlev * max(0.7, 1.0 - 0.15 * f_spr)
        
    return dlev

# ==================== 4. 纯彩虹面 (Rainbow) ====================
def correct_board_4_rainbow(dlev, position, bucket, draw, spr):
    f_spr = get_spr_factor(spr)
    
    # 数据表明：OESD 大顺子听牌原始 DLEv 严重低估 (DL 28.7 -> Full 43.0)
    # SPR 越深，隐含赔率越高，但必须收敛
    if draw == "OESD":
        return dlev * (1.0 + 0.28 * f_spr) # 在 SPR=20 时乘数约为 1.45，完美契合数据
        
    # 顶对和中等成牌在干燥面上略微虚高，全树里会被对子以上动作挤压
    if bucket in ["桶2:中等顶对", "桶3:中等成牌"]:
        return dlev * max(0.8, 1.0 - 0.08 * f_spr)
        
    return dlev

# ==================== 5. 两同花面 A (Two-Tone Type A) ====================
def correct_board_5_twotone_a(dlev, position, bucket, draw, spr):
    f_spr = get_spr_factor(spr)
    
    # 顶对在高张多听面上后劲不足（反向隐含赔率凸显，原方案直接减到30%属于崩溃，现修正）
    if bucket == "桶2:中等顶对":
        return dlev * max(0.75, 1.0 - 0.10 * f_spr)  # SPR=20时衰减到约 84% 
        
    # 强同花听牌变现良好
    if draw == "FlushDraw":
        return dlev * (1.0 + 0.12 * f_spr)
        
    return dlev

# ==================== 6. 两同花面 B (Two-Tone Type B) ====================
def correct_board_6_twotone_b(dlev, position, bucket, draw, spr):
    f_spr = get_spr_factor(spr)
    
    # 低张两同花面上，由于悬挂的高张更多，顶对（如Q）被反超的概率更大
    if bucket == "桶2:中等顶对":
        return dlev * max(0.7, 1.0 - 0.12 * f_spr)
        
    if draw == "FlushDraw":
        return dlev * (1.0 + 0.15 * f_spr)
        
    return dlev

# ==================== 7. 单调面 A (Monotone Type A) ====================
def correct_board_7_monotone_a(dlev, position, bucket, draw, spr):
    f_spr = get_spr_factor(spr)
    
    # 复合听牌（FDPlusSD）长线极强
    if draw == "FDPlusSD":
        return dlev * (1.0 + 0.15 * f_spr)
        
    # 纯空气在已经成花的单色面上毫无价值
    if bucket == "桶7:纯空气":
        return dlev * max(0.5, 1.0 - 0.25 * f_spr)
        
    return dlev

# ==================== 8. 单调面 B (Monotone Type B) ====================
def correct_board_8_monotone_b(dlev, position, bucket, draw, spr):
    f_spr = get_spr_factor(spr)
    
    # 你的核心痛点数据发现：低张同花面上，非同花超强成牌（暗三条66）FullEV(59.2) 远高于 DLEv(50.9)
    # 因为深筹码场景下，暗三条一旦买到葫芦，拥有将对手“已成小同花”连根拔起的绝对隐含赔率！
    if bucket == "桶1:超强成牌" and draw == "None":
        return dlev * (1.0 + 0.10 * f_spr) # SPR=20时约放大1.16倍，完美补回少算的 8.28 点
        
    # 同花听牌（特别是坚果FD）在低张面上极具统治力
    if draw == "FlushDraw":
        return dlev * (1.0 + 0.12 * f_spr)
        
    return dlev

```
