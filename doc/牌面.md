在德州扑克中，“同构翻牌”（Isomorphic Flops）是指**在不考虑具体花色的情况下，纯粹基于牌面“花色结构”和“点数成对情况”而归纳出的战略等价分类**。

通过去除花色差异（例如，红桃A-红桃K-红桃Q 与 方块A-方块K-方块Q 在策略上完全一样），德州扑克中总共 **22,100** 种可能的翻牌牌面，可以被完美精简为 **1,755 种独立同构翻牌**。

在这 1,755 种同构翻牌中，如果纯粹按“公牌的对子情况”**与**“公牌的花色分布（彩虹/两同花/单色）”两个维度交织组合，就会演化出经典的 **8 种核心基本同构牌面分类**。这是 GTO 软件（如 PioSolver、GTO Wizard）在对翻牌进行底层逻辑分类时最基础的 8 种形态：

---

### 1. 三条牌面 (Trips Boards)

翻牌的三张公牌点数完全相同（例如：$A\spadesuit A\heartsuit A\diamondsuit$、$8\clubsuit 8\spadesuit 8\heartsuit$）。

* **1. 散色三条 (Trips)**：由于三张牌点数相同，必然包含三种不同的花色（即彩虹三条）。

> **战术特点：** 极度干燥，由于公牌已经拿走了三张相同的牌，任何人拿到第四张形成四条（Quad）或者成葫芦（Full House）的概率极低。

### 2. 对子牌面 (Paired Boards)

翻牌中有两张牌点数相同，第三张牌不同（例如：$A A K$ 或 $7 7 2$）。根据第三张牌的花色，分为两种：

* **2. 对子彩虹面 (Paired Rainbow)**：三张牌包含三种不同的花色（例如：$K\spadesuit K\heartsuit 5\diamondsuit$）。
* **3. 对子两同花面 (Paired Two-Tone)**：对子本身或第三张牌带有两张同花（例如：$K\spadesuit K\spadesuit 5\diamondsuit$ 不可能，实际为 $K\spadesuit K\heartsuit 5\spadesuit$ 或 $K\spadesuit 5\spadesuit 2\heartsuit$ 中包含两张同花）。

> **战术特点：** 翻牌前激进者（Preflop Aggressor）通常在对子面拥有极大的范围优势，尤其是高对子面（如 A-A-x），非常适合进行高频率的小尺寸持续下注（C-bet）。

### 3. 三张杂牌面 / 无对子面 (Unpaired / High Card Boards)

三张公牌的点数各不相同。这是德州扑克中最常见的翻牌形态，根据花色的分布，它被严格拆分为以下 5 种同构：

* **4. 纯彩虹面 (Rainbow - 3 Suits)**：三张牌属于三种完全不同的花色（例如：$A\spadesuit K\heartsuit 4\diamondsuit$）。
* *特点：* 没有任何同花听牌的可能，属于偏干燥的桌面。


* **5. 两同花面 A (Two-Tone Type A)**：有两张牌同花，且这两张同花牌包括了翻牌里**较大**的那张牌（例如：$A\spadesuit K\spadesuit 4\heartsuit$）。
* **6. 两同花面 B (Two-Tone Type B)**：有两张牌同花，但这两张同花牌是翻牌里**较小**的两张牌（例如：$A\heartsuit K\spadesuit 4\spadesuit$）。
* *为什么要分A/B？* 因为在 GTO 计算中，大牌同花（包含 A 或 K 的同花听牌）与小牌同花的范围重叠度、阻挡牌（Blocker）效应完全不同，策略有明显差异。


* **7. 单色面 / 纯同花面 A (Monotone Type A)**：三张牌全部为同一花色，且包含了该花色中的**绝对高牌**（如 $A\spadesuit K\spadesuit T\spadesuit$）。
* **8. 单色面 / 纯同花面 B (Monotone Type B)**：三张牌全部为同一花色，但属于**中低张同花**（如 $9\spadesuit 7\spadesuit 4\spadesuit$）。
* *特点：* 单色面（Monotone）极为湿润，场上可能已经有人直接成花，或者单张花色牌就能听花。



---

### 总结：8 种同构的结构图解

为了更直观地理解，我们可以通过下表看清这 8 种分类是如何将任何德州扑克翻牌收入囊中的：

| 编号 | 牌面大类 | 花色具体结构描述 | 典型举例 (Isomorphic Base) |
| --- | --- | --- | --- |
| **1** | **三条面** | 3张点数相同 (必然三花色) | $A\spadesuit A\heartsuit A\diamondsuit$ |
| **2** | **对子面** | 2张点数相同 + 3种不同花色 (彩虹) | $K\spadesuit K\heartsuit 5\diamondsuit$ |
| **3** | **对子面** | 2张点数相同 + 包含2张同花 | $K\spadesuit K\heartsuit 5\spadesuit$ |
| **4** | **无对子** | 3张点数不同 + 3种不同花色 (彩虹) | $Q\spadesuit J\heartsuit 4\diamondsuit$ |
| **5** | **无对子** | 3张点数不同 + 2张同花（含大牌） | $Q\spadesuit J\spadesuit 4\heartsuit$ |
| **6** | **无对子** | 3张点数不同 + 2张同花（仅小牌） | $Q\heartsuit J\spadesuit 4\spadesuit$ |
| **7** | **无对子** | 3张点数不同 + 3张全同花（高牌面） | $A\spadesuit J\spadesuit 9\spadesuit$ |
| **8** | **无对子** | 3张点数不同 + 3张全同花（低牌面） | $8\spadesuit 6\spadesuit 3\spadesuit$ |

在日常复盘或使用 Solver（求解器）时，理解这 8 种基础同构能帮你快速建立起“牌面结构”的概念，而不用死记硬背成千上万种具体的花色组合。
