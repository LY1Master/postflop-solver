# v3 修正公式与偏差分析

## 修正计算方式

每条修正都是一个完整的数学表达式，把 SPR 代入后直接算出修正后 DLEv：

```
修正后 DLEv = DLEv × (一段关于 SPR 的算式)
```

**核心函数（所有牌面共用）：**
```
内部计算 t = 2 × SPR / (SPR + 5)

SPR=5 时  t = 2×5/(5+5)   = 10/10 = 1.0
SPR=20 时 t = 2×20/(20+5) = 40/25 = 1.6
```

## 牌面 #1：三条面 (8h8s8d)

### 桶1:超强成牌 / 桶2 / 桶4 / 桶5
**不做修正**
```
修正后 DLEv = DLEv
```

### 桶3:中等成牌
```
修正后 DLEv = DLEv × max(0.5, 1 - 0.12 × 2×SPR/(SPR+5))

SPR=5:  修正后 DLEv = DLEv × max(0.5, 1 - 0.12 × 1.0) = DLEv × 0.88
SPR=20: 修正后 DLEv = DLEv × max(0.5, 1 - 0.12 × 1.6) = DLEv × 0.81
```

### 桶6:弱听牌 / 桶7:纯空气
```
修正后 DLEv = DLEv × max(0.1, 1 - 0.45 × 2×SPR/(SPR+5))

SPR=5:  修正后 DLEv = DLEv × max(0.1, 1 - 0.45 × 1.0) = DLEv × 0.55
SPR=20: 修正后 DLEv = DLEv × max(0.1, 1 - 0.45 × 1.6) = DLEv × 0.28
```
